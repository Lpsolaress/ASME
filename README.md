## Descripción del Proyecto
ASME Industrial Precision es una solución integral para la digitalización de estudios de tiempos y movimientos bajo la normativa ASME. Utiliza Inteligencia Artificial para transformar descripciones de lenguaje natural en datos estructurados de ingeniería, permitiendo optimizar procesos industriales con una precisión sin precedentes.

## Flujo de Trabajo (5 Pasos)
1.  **Configuración**: Registro de la organización y parámetros operativos.
2.  **Captura**: Grabación de voz o texto para documentar actividades.
3.  **Identificación**: Clasificación automática entre Valor Añadido (VA) y Desperdicio (NVA).
4.  **Análisis**: Dashboard de ingeniería con proyecciones de ahorro y hojas de ruta.
5.  **Reporte**: Generación de informes PDF certificados con analítica visual.

## Características Principales
*   **Transcripción Inteligente**: Procesamiento de voz para una captura de datos manos libres en planta.
*   **Análisis Predictivo**: La IA sugiere mejoras específicas (RPA, Vision AI, IoT) basadas en la carga anual.
*   **Reportes de Grado Industrial**: Documentos PDF diseñados para presentaciones ejecutivas y auditorías.
*   **Diseño Técnico**: Interfaz optimizada para entornos industriales con alto contraste y claridad de datos.
